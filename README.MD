# Orbita

O **Orbita** é um projeto pessoal que estou desenvolvendo para estudar dinâmica e controle de atitude de satélites.

A ideia não é simplesmente desenhar um cilindro (ou um satélite) girando em OpenGL. O objetivo é simular, da forma mais próxima possível, como um satélite realmente se comporta no espaço.

O projeto ainda está no começo e muita coisa vai mudar ao longo do desenvolvimento. Conforme ele evoluir, pretendo adicionar modelos físicos cada vez mais realistas, incluindo:

* dinâmica rotacional;
* momentos de inércia;
* perturbações e ruídos;
* controle por propulsores;
* diferentes algoritmos de controle;
* visualização 3D em tempo real.

Não tenho a pretensão de fazer um simulador profissional, mas sim um projeto que seja interessante do ponto de vista de engenharia e que me permita estudar os conceitos usados em sistemas reais.
